### Customization * Get source code from [GitHub](https://github.com/ASaiM/framework) ``` $ git clone https://github.com/ASaiM/framework.git $ cd framework ``` * Install the required dependencies ``` $ ./src/install_dependencies.sh ``` * Configure the environment ``` $ ./src/configure.sh ``` * Launch the instance ``` $ ./src/launch_asaim.sh ``` * Browse it on [http://0.0.0.0:8080/](http://0.0.0.0:8080/)
